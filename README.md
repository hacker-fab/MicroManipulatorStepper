# HackerFab Nanopositioner

This repository contains **HackerFab’s extended version** of the open-source nanopositioner originally developed by Diffraction Limited.

We build directly on the original design and contribute additional capabilities, including:
- Rotational control
- Integrated heating
- Vacuum compatibility
- Ongoing improvements for lab use at HackerFab

## Credit

All core mechanical, electrical, and firmware design work is credited to the original project:

https://github.com/0x23/OpenMicroManipulator  

This repo is a derivative work that extends their platform for new use cases with additional custom made components.
